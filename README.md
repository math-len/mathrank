### Module Convention
- 하위 모듈은 항상 `mathrank-`을 prefix 로 가지도록 한다. ex) `mathrank-member`
- 다음과 같이 연장한다. `mathrank-member-read`, `mathrank-member-write`, ...
- 자바 패키지 경로는 항상 `kr.co.mathrank`로 시작한다.
